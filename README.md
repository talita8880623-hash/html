# README
## Aqui vai uma breve explição de cada página criada.  
*Nesse site `index.html` eu criei uma página inicial para apresentar
 meu primeiro projeto. Nela, eu dou boas-vindas aos visitantes e explico que o site
  contém minhas informações pessoais e minhas receitas favoritas. Também incluí links
   para outras páginas, onde é possível acessar mais detalhes sobre mim e ver as receitas 
   que eu gosto.*

*nesse site `sobre.mim`   Este site foi criado para apresentar um pouco sobre mim, meus
 gostos,
hobbies e interesses. Aqui você poderá conhecer mais sobre minha
personalidade, os estilos de música que eu gosto, minhas séries favoritas
 e algumas atividades que fazem parte da minha rotina, como academia e vôlei.*

 *Nesse site `receitas.html` aqui estao minhas receitas e ensina como preparar um delicioso     bolo de cenoura com cobertura de chocolate.
 O passo a passo é simples e fácil de entender, ideal para quem gosta de fazer sobremesas
 caseiras.
 Além disso, a receita utiliza ingredientes comuns e mostra desde o preparo da massa até a cobertura.
 O objetivo desta página é apresentar uma receita saborosa e prática, perfeita para compartilhar
 com a família e os amigos em diferentes momentos.*
    
## Encerramento 
        Este site foi criado com o objetivo de apresentar um pouco sobre mim, meus gostos,
        interesses e também compartilhar conteúdos simples e criativos, como a receita de
        bolo de cenoura. Durante o desenvolvimento da página, foi possível organizar informações,
        aprender mais sobre HTML e criar um espaço personalizado e informativo.
